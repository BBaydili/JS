Google Forms to Google Sheets

A simple, public database to save your data in the cloud

The method described below is only for teaching purposes -- All data will be public

1. Make a Google Form

Visit docs.google.com/forms to make a new form. Suppose we want a data table with 3 fields: user, topic, marks. All fields will be "short answer". Another field for timestamp will be added by the server. We are ready to add data.

2. Get the data-entry address

Enter the field names as the answer to all three questions in the form. Go to the three-dots menu to select "pre-filled link" and copy the address:

We need the magic numbers hidden in this link in order to make a proper data entry.

We need to replace viewform => formResponse in the code:

const FORM_URL = "https://docs.google.com/forms/d/e/"
    +"1FAIpQLScbLFzsO3ugLVVO8TWS28pmsvrGnf9yyFyYv4UIK8tncCA_NA/"
    +"formResponse?usp=pp_url"
    +"&entry.1395280695="+user
    +"&entry.387807091="+topic
    +"&entry.1944284271="+marks
//  +"&submit=Submit" not needed

//put the link on an iframe and use it to post one row to the table

3. Move the data to Google Sheets

The default location of the data is Google Drive. It should be moved to Google Sheets, where it will be available for the clients.

Clik on Responses and "Create Spreadsheet" -- Your new sheet wil be opened

4. Publish your data

Select in the File menu "Publish to web" and copy the address. This link gets the data in human-readable HTML format. Other formats such as JSON, CSV, and TSV are also available:

const DOCS_URL = "https://docs.google.com/spreadsheets/u/1/d/e/2PACX-"
    +"1vStbPfY8oQuJKDQwIiFwRMs5V6loOPmr3gzc8UQN36z4cbfLZuPedXePhgq9053P7KsDA8YZ42cvBVE/"
    +'pub?output=tsv' //tab-separated values

Summary: One link adds a row to the table, the other link reads the data



If the table becomes too large, unused rows can be deleted manually from the sheet. The data is read-only, nobody can modify it except the owner. But new rows can be added anyone who has the link. Forms data is still in the Drive, not used by the Sheets. So it may be deleted once-in-a-while to save space.
